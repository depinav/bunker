Bunker Help

You can get detailed help by typing /help in the chat message box. This will invoke some private help messages that only you will see. Otherwise, if you prefer this pretty help pop-up, a quick summary of commands is below:

Commands

Command Description Example
/nick [nickname] Change your nickname. /nick kindohm
/me [action] Make your nick do something in the room. /me does something cool
/roll [number] Roll a die. /roll 20
/magic8ball [optional question] The magic 8 ball will return 1 of 20 possible answers /magic8ball Will it rain today?
/hangman [letter guess] | /h [letter guess] Play a game of hangman /hangman e

Admin Commands

Command Description Example
/topic [topic name] Change the room topic. /topic fish

Images

Want to upload an image to a chat room? Just drag-n-drop from your desktop, or copy image data from your desktop and paste into the chat message input box.

Gravatar

Want a custom avatar icon next to your name in the chat room members list? Create a gravatar at gravatar.com associated with the same email address you logged in with.

Text Formatting

Format Character Description Example Result
Bold * Create text with bold emphasis. my *cat* eats fish my cat eats fish
Italics _ Create text with italic emphasis. my _cat_ eats fish my cat eats fish
Spoilers | Hide text, as to not reveal too much. |my cat eats fish| my cat eats fish
Strikethrough ~ Create text with strikethrough styling. my ~cat~ eats fish my cat eats fish
